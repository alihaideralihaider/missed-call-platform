# SaanaOS Brand Kit v0.1

This folder is a starting brand identity kit for SaanaOS. It includes:

- Logo concept board
- Icon-only cloche/circuit logo
- Starter SVG icon
- Food-tech pattern illustration
- Missed-call recovery ad concepts in common social sizes
- Working color palette
- Brand guidelines PDF and DOCX

Important: AI-generated PNGs are useful for direction and marketing mockups. For final commercial use, rebuild the logo and text-heavy ad creatives as true vector/editable files in Figma, Illustrator, Canva, or code.
